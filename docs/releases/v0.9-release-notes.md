# Overview

Version 0.9 introduces the first major progression systems to Last Man Standing.

Players can now unlock achievements, earn XP, view detailed statistics and share public player profiles. This release also includes numerous gameplay improvements and bug fixes ahead of the first closed alpha.

---

# New Features

## Achievements

A new achievement system has been introduced.

Features include:

- Visible achievements
- Discovery achievements
- Achievement XP rewards
- Progress tracking
- Achievement notifications
- Rarest achievement display

---

## Player Profiles

Players now have public profiles.

Features include:

- Public profile page
- Optional profile privacy
- Public achievement showcase
- Hidden achievement protection

---

## Player Statistics

Player profiles now include statistics such as:

- Competitions entered
- Competitions completed
- Competitions won
- Competitions currently alive
- Competition win percentage
- Successful picks
- Eliminated picks
- Weeks survived
- Joker uses
- Joker saves
- Unique clubs picked
- Favourite club

---

## Achievement Notifications

Achievement unlocks are now stored until viewed.

Notifications can be dismissed and will only be shown once.

---

# Gameplay Improvements

- Competition winner achievements
- Joint winner achievements
- Improved result processing
- Gameweek processing tracking
- Better handling of competition completion

---

# Bug Fixes

- Players who fail to submit a selection before the deadline are now eliminated correctly.
- Improved visibility of login error messages.
- Various stability improvements and internal code clean-up.

---

# Internal Improvements

Although not directly visible to players, this release includes significant internal improvements including:

- Achievement service architecture
- Modular achievement checkers
- Statistics service
- Improved result processing
- Better notification handling
- Additional automated testing
